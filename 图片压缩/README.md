# 网页开发项目说明文档

## 项目概述
这是一个优雅的在线图片压缩工具网站。用户可以上传图片，预览压缩效果并下载压缩后的图片。网站采用苹果风格设计，简洁美观且易于使用。

### 核心功能
- 支持上传PNG、JPG等格式图片
- 支持自定义压缩比例
- 实时预览压缩前后的图片效果
- 显示压缩前后的文件大小
- 支持下载压缩后的图片

### 设计风格
- 采用苹果风格设计
- 简约优雅的界面
- 柔和的动画效果
- 清晰的视觉反馈
